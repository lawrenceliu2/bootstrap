Buttons

You can create buttons of different styles with the <button> tag. To set a style, add the Bootstrap class btn-<STYLE>.

Example Code:

<button type="button" class="btn btn-default">Default</button>

Tables

Tables are created with the default <table> class. To enable bootstrap tables, add the class table. To set a style, add class table-<STYLE>. Another fun thing to add is class table-hover, which changes the color of rows as you hover over it.

To have a bold first row and seperator, use the <thead> tag to enclose the first row your of your <table>. In addition, for headers, you must use the <th> tag within your <tr>s. instead of the default <td>. The rest of your table should be enclosed in a <tbody> tag.
