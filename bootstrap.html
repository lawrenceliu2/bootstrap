Text

Text is mostly unchanged in Bootstrap.

You can highlight text with <mark>words</mark>.

And you can delete lines by using <del>words</del>.

You can insert quotes with <blockquote>words</blockquote>.

And you can credit people too by adding <footer>A person <cite>on a day</cite></footer>.

Images

Images are mostly the same in Bootstrap too.

Use the <img> tag with scr="potato.png" as usual. But by adding class="img-responsive", the image changes size depending on its container and your screen size.

<img src="http://www.potato.es/wp-content/uploads/potato-slider.png" class="img-responsive">

Buttons

You can create buttons of different styles with the <button> tag. To set a style, add the Bootstrap class btn-<STYLE>.

Example Code:

<button type="button" class="btn btn-default">Default</button>

Tables

Tables are created with the default <table> class. To enable bootstrap tables, add the class table. To set a style, add class table-<STYLE>. Another fun thing to add is class table-hover, which changes the color of rows as you hover over it.

To have a bold first row and seperator, use the <thead> tag to enclose the first row your of your <table>. In addition, for headers, you must use the <th> tag within your <tr>s. instead of the default <td>. The rest of your table should be enclosed in a <tbody> tag.

Example Code:

<table class="table table-default table-hover">
   <thead>
      <tr>
         <th>Name</th>
         <th>ID</th>
         <th>Password</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>Lawrence</td>
         <td>673647859</td>
         <td>2jk45v2j3v5</td>
      </tr>
   </tbody>
</table>

Forms

Instead of using <form>, in Bootstrap we use <input>.

We can change the <type> of the input to determine whether its a text field for emails, passwords, checkbox, radio button, etc.

Example Code:

<form>
<div class="form-group">
   <label for="emailz">Email address</label>
   <input type="email" class="form-control" id="emailz" placeholder="Email">
</div>
<div class="form-group">
   <label for="pw">Password</label>
   <input type="password" class="form-control" id="pw" placeholder="Password">
</div>
<button type="submit" class="btn btn-default">Submit</button>
</form>
